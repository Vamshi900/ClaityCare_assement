# Oscar App

See the [main README](../README.md) at the repo root for full documentation, setup instructions, and architecture overview.

## Quick Reference

```bash
./setup.sh          # One-command setup
make start          # Start services
make stop           # Stop services
make status         # Check pipeline stats
make extract        # Run extraction (10 policies)
make logs           # Tail backend logs
make db-reset       # Wipe and recreate database
```
